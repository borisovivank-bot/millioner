# Победи миллион (Win the Million)

Это консольная игра-викторина, написанная на Python.

## Описание

Проект представляет собой игру, в которой пользователю задаются вопросы, и он должен выбрать правильный ответ из предложенных вариантов. Игра напоминает классические телевизионные викторины.

## Как запустить

Для запуска игры вам потребуется установленный Python.

1. Скачайте код проекта.
2. Запустите файл `main.py` через терминал:

```bash
python main.py
```

## Текущее состояние

Проект находится в стадии разработки. Реализован базовый вывод вопросов и проверка ввода.
